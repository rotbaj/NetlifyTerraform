About This Project

This project demonstrates a fully automated and secure deployment pipeline for a static website. By leveraging Terraform, we can codify our infrastructure as code, ensuring that every deployment is consistent and repeatable. This approach minimizes manual configuration errors and allows for easy version control of the entire infrastructure.

Services & Technologies Used

  • **Terraform:** For managing infrastructure as code.
  • **Netlify:** For continuous deployment and hosting.
  • **GitHub:** As the source code repository.
  • **HCP Terraform:** For secure remote state management.
  • **HTML & CSS:** For the front-end design.
